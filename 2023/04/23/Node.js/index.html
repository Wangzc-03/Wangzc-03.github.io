Node.js

一、首先了解Javascript如何在浏览器中运行(以Chrome浏览器为例)

其运行需要

  1. 运行环境(浏览器本身 )
  2. JavaScript解析引擎（Chrome为V8引擎，性能较好）
  3. 浏览器内置API函数

如图，基础的Js代码在浏览器运行环境中调用其内置的API函数，然后被解析引擎解析并运行出结果。

由此可见js的实现需要运行环境和解析引擎两个必要条件

二、Node.js是什么？

2009年5月，一个名为Ryan Dahl的人将Chrome浏览器的V8引擎与C++代码结合开发出了Node.js，使得js可以脱离浏览器环境运行。

所以Node.js是一个基于Chrome V8引擎的Javascript运行环境

三、如何安装、检查

1、打开官网选择左侧稳定版下载，安装一路next，不建议更改安装路径

2、按下win+r打开终端输入node -v回车可以看到安装的版本号，注意杠前有一个空格。可以直接到文件所处位置，在路径处输入cmd直接打开终端并调整路径。

四、使用

1、使用Node运行js代码：

想要运行本地的js代码，需要先调整终端所处的目录，输入cd+空格+文件路径回车即可调整当前路径。

五、包和npm

1、包是什么？

包是node.js中的第三方模块

虽然Node.js的内置模块提供了一些底层的API,但基于内置模块进行开发时，效率低 。使用第三方团队基于内置模块封装出来的API更加方便高效

2、npm是什么

npm是一个包管理工具，npm在下载node.js已经同时安装

输入npm -v查看安装的npm的版本号

五、npm

1、使用npm安装包

输入npm install 包的名字并回车 ，可以简写为 npm i 包的名字，使用@加版本号下载指定版本

eg：npm i mark@4.1.2
